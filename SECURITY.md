# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✓         |

## Reporting a Vulnerability

Please report security vulnerabilities via GitHub's
[private vulnerability reporting](https://github.com/MartinHoblisch/TRACLUS/security/advisories/new)
or by email to m.hoblisch@gmail.com.

Expected response: acknowledgement within 7 days.
We aim to release a fix within 30 days for confirmed vulnerabilities.
