# Leaflet map for TRACLUS objects

**\[stable\]**

## Usage

``` r
tc_leaflet(x, ...)

# S3 method for class 'tc_trajectories'
tc_leaflet(x, ...)

# S3 method for class 'tc_partitions'
tc_leaflet(x, show_points = TRUE, ...)

# S3 method for class 'tc_clusters'
tc_leaflet(x, ...)

# S3 method for class 'tc_representatives'
tc_leaflet(x, show_clusters = FALSE, ...)

# S3 method for class 'tc_traclus'
tc_leaflet(x, ...)
```

## Arguments

- x:

  A TRACLUS workflow object with `coord_type = "geographic"`.

- ...:

  Additional arguments (method-specific, see individual methods).

- show_points:

  Logical; if `TRUE` (default for `tc_partitions`), characteristic
  points are drawn as black cross markers at segment endpoints (matching
  the base [`plot()`](https://rdrr.io/r/graphics/plot.default.html)
  style).

- show_clusters:

  Logical; if `FALSE` (default), cluster member segments are drawn in
  grey and only representatives are highlighted in colour. If `TRUE`,
  cluster segments are drawn in full colour. See
  [`plot.tc_representatives()`](https://martinhoblisch.github.io/TRACLUS/reference/plot.TRACLUS.md)
  for details.

## Value

A `leaflet` htmlwidget object (can be further customized via piping,
e.g., `tc_leaflet(x) |> leaflet::setView(...)`).

## Details

Creates interactive Leaflet maps for geographic TRACLUS data. Only
available when `coord_type = "geographic"` — euclidean data will produce
an error. Requires the `leaflet` package.
