# Resolve the distance method from coord_type and user input

Resolve the distance method from coord_type and user input

## Usage

``` r
.resolve_method(coord_type, method, verbose)
```

## Arguments

- coord_type:

  Character: "euclidean" or "geographic".

- method:

  Character or NULL.

- verbose:

  Logical.

## Value

Resolved method string.
