# Compute cross-track distance from point to great circle (C++)

Compute cross-track distance from point to great circle (C++)

## Usage

``` r
.cpp_cross_track_dist(plon, plat, lon1, lat1, lon2, lat2)
```

## Arguments

- plon, plat:

  Point coordinates in degrees

- lon1, lat1, lon2, lat2:

  Great circle endpoints in degrees

## Value

Absolute cross-track distance in meters (double)
