# R reference: cross-track distance (absolute, meters)

R reference: cross-track distance (absolute, meters)

## Usage

``` r
.r_cross_track(p, a, b)
```

## Arguments

- p:

  Point as c(lon, lat) in degrees.

- a:

  Start of great circle as c(lon, lat).

- b:

  End of great circle as c(lon, lat).

## Value

Absolute cross-track distance in meters (double).
