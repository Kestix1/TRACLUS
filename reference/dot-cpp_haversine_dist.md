# Compute haversine distance between two points (C++)

Compute haversine distance between two points (C++)

## Usage

``` r
.cpp_haversine_dist(lon1, lat1, lon2, lat2)
```

## Arguments

- lon1, lat1:

  Coordinates of point 1 in degrees

- lon2, lat2:

  Coordinates of point 2 in degrees

## Value

Distance in meters (double)
