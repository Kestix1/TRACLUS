# Compute initial bearing between two points (C++)

Compute initial bearing between two points (C++)

## Usage

``` r
.cpp_initial_bearing(lon1, lat1, lon2, lat2)
```

## Arguments

- lon1, lat1:

  Coordinates of point 1 in degrees

- lon2, lat2:

  Coordinates of point 2 in degrees

## Value

Bearing in degrees \[0, 360) (double)
