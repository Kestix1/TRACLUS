# R reference: initial bearing (forward azimuth) in degrees \[0, 360)

R reference: initial bearing (forward azimuth) in degrees \[0, 360)

## Usage

``` r
.r_bearing(p1, p2)
```

## Arguments

- p1:

  Point 1 as c(lon, lat) in degrees.

- p2:

  Point 2 as c(lon, lat) in degrees.

## Value

Bearing in degrees (double).
