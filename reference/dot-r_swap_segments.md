# R reference: apply swap convention — returns list(si, ei, sj, ej) with segment i being the longer one.

R reference: apply swap convention — returns list(si, ei, sj, ej) with
segment i being the longer one.

## Usage

``` r
.r_swap_segments(si, ei, sj, ej)
```

## Arguments

- si, ei, sj, ej:

  Segment endpoints (each length-2 numeric).

## Value

List with elements si, ei, sj, ej (possibly swapped).
