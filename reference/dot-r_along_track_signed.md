# R reference: signed along-track distance (meters)

R reference: signed along-track distance (meters)

## Usage

``` r
.r_along_track_signed(p, a, b)
```

## Arguments

- p:

  Point as c(lon, lat) in degrees.

- a:

  Start of great circle as c(lon, lat).

- b:

  End of great circle as c(lon, lat).

## Value

Signed along-track distance in meters (double).
