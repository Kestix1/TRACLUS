# R reference: project point onto LINE through ls-le

R reference: project point onto LINE through ls-le

## Usage

``` r
.r_project_onto_line(p, ls, le)
```

## Arguments

- p:

  Point (length-2 numeric).

- ls:

  Line start (length-2 numeric).

- le:

  Line end (length-2 numeric).

## Value

Scalar projection parameter t.
