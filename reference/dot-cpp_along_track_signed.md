# Compute signed along-track distance (C++)

Compute signed along-track distance (C++)

## Usage

``` r
.cpp_along_track_signed(plon, plat, lon1, lat1, lon2, lat2)
```

## Arguments

- plon, plat:

  Point coordinates in degrees

- lon1, lat1, lon2, lat2:

  Great circle endpoints in degrees

## Value

Signed along-track distance in meters (double)
