# R reference: perpendicular distance from point to LINE

R reference: perpendicular distance from point to LINE

## Usage

``` r
.r_point_to_line_dist(p, ls, le)
```

## Arguments

- p:

  Point (length-2 numeric).

- ls:

  Line start (length-2 numeric).

- le:

  Line end (length-2 numeric).

## Value

Perpendicular distance (double).
