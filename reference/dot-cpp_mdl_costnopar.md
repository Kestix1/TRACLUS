# Compute MDL cost of no-partitioning (costnopar) for testing

Compute MDL cost of no-partitioning (costnopar) for testing

## Usage

``` r
.cpp_mdl_costnopar(sub_x, sub_y, method)
```

## Arguments

- sub_x:

  Numeric vector of point x-coordinates.

- sub_y:

  Numeric vector of point y-coordinates.

- method:

  Integer: 0 for euclidean, 1 for haversine.

## Value

The MDL costnopar value (double).
