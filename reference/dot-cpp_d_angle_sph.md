# Compute spherical angle distance between two segments (C++)

Compute spherical angle distance between two segments (C++)

## Usage

``` r
.cpp_d_angle_sph(
  lon_si,
  lat_si,
  lon_ei,
  lat_ei,
  lon_sj,
  lat_sj,
  lon_ej,
  lat_ej
)
```

## Arguments

- lon_si, lat_si, lon_ei, lat_ei:

  Segment i endpoints in degrees

- lon_sj, lat_sj, lon_ej, lat_ej:

  Segment j endpoints in degrees

## Value

Angle distance in meters (double)
