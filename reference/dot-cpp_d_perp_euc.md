# Compute euclidean perpendicular distance between two segments (C++)

Compute euclidean perpendicular distance between two segments (C++)

## Usage

``` r
.cpp_d_perp_euc(si_x, si_y, ei_x, ei_y, sj_x, sj_y, ej_x, ej_y)
```

## Arguments

- si_x, si_y, ei_x, ei_y:

  Coordinates of segment i

- sj_x, sj_y, ej_x, ej_y:

  Coordinates of segment j

## Value

Perpendicular distance (double)
